
## 스프링부트와 JPA 활용
spring boot + jpa web application

***

### 프로젝트 생성
- 스프링 부트 스타터(https://start.spring.io)
- 사용 기능 : web, thymeleaf, jpa, h2, lombok, validation
- java 11

***

### 웹 애플리케이션 개발
- 프로젝트 환경설정
- 요구사항 분석
- 도메인과 테이블 설계
- 아키텍처 구성
- 핵심 비즈니스 로직 개발(회원, 상품, 주문)
- 테스트
- 웹 계층 개발

***

### 성능 최적화
- JPA 극한의 조회 성능 최적화
- 6단계로 성능 개선
